Configuring TortoiseGit for Graphical Diff
вҖӢ 1. Right-click in Windows Explorer to expose the TortoiseGit options
вҖӢ 2. Mouse-over вҖҳTortoiseGit>> SettingsвҖҷ
вҖӢ 3. In the list of settings on the left, select вҖҳDiff ViewerвҖҷ,On the right side of the dialog, select вҖҳAdvancedвҖҷ
вҖӢ 4. When the вҖҳAdvanced diff settingsвҖҷ dialog appears, select вҖҳAddвҖҷ
вҖӢ 5. In the dialog that appears, type .vi as the extension
вҖӢ 6. Where it prompts you for the path to the external program, type the following: вҖңC:\Program Files\National Instruments\Shared\LabVIEW Compare\LVCompare.exeвҖқ %mine %base -nobdcosm -nobdpos
вҖӢ 7. Repeat this operation for the file-type .ctl
Configuring TortoiseGit for Graphical Merge
вҖӢ 1. Right-click in Windows Explorer to expose the TortoiseGit options
вҖӢ 2. Mouse-over вҖҳTortoiseGit>> SettingsвҖҷ
вҖӢ 3. In the list of settings on the left, select вҖҳMerge ToolвҖҷ under вҖҳDiff ViewerвҖҷ
вҖӢ 4. On the right side of the dialog, select вҖҳAdvancedвҖҷ
вҖӢ 5. When the вҖҳAdvanced merge settingsвҖҷ dialog appears, select вҖҳAddвҖҷ
вҖӢ 6. In the dialog that appears, type .vi as the extension
вҖӢ 7. Where it prompts you for the path to the external program, type the following: вҖңC:\Program Files\National Instruments\Shared\LabVIEW Merge\LVMerge.exeвҖқ %base %mine %theirs %merged
вҖӢ 8. Repeat this operation for the file-type .ctl

еҸҲеҲ°дәҶ5жңҲд»ҪдёҖе№ҙдёҖеәҰзҡ„NI Week,LabVIEW 2018жҢүжңҹжӯЈејҸеҸ‘еёғ,еҖјеҫ—жіЁж„Ҹзҡ„жңүдёӨзӮ№,

Pythonи°ғз”Ёж”ҜжҢҒ

NXGйЈҺж јзҡ„жҺ§д»¶

жңүдёӘеӨ§еқ‘,LabVIEWи°ғз”Ёзҡ„жҳҜPython36.dll,еҜјиҮҙ32bitзҡ„LabVIEWеҸӘиғҪи°ғ32bitзҡ„Python,еҸӘжңү64bitзҡ„LabVIEWжүҚиғҪи°ғз”Ё64bitзҡ„Python;з»•иҝҮзҡ„еҠһжі•жҳҜзӣҙжҺҘз”Ёе‘Ҫд»ӨиЎҢжҺҘеҸЈи°ғз”ЁPython,дёҚиҝҮжҜҸжқЎе‘Ҫд»ӨиҝҗиЎҢж—¶йғҪиҰҒжү“ејҖPython.exe,иҝҗиЎҢе®Ңд№ӢеҗҺдјҡе…ій—ӯPython.

дә’иҝһжҺҘеҸЈйҖүжқҝж–°еўһPythonеӯҗйҖүжқҝпјҢеҸҜдҪҝз”Ёе®ғд»ҺLabVIEWд»Јз Ғдёӯи°ғз”ЁPythonд»Јз ҒгҖӮPythonйҖүжқҝеҢ… еҗ«д»ҘдёӢеҮҪж•°пјҡ

вҖў жү“ејҖPythonдјҡиҜқпјҚз”Ёзү№е®ҡзүҲжң¬зҡ„Pythonжү“ејҖPythonдјҡиҜқгҖӮ

вҖў PythonиҠӮзӮ№пјҚзӣҙжҺҘи°ғз”ЁPythonеҮҪж•°гҖӮ

вҖў е…ій—ӯPythonдјҡиҜқпјҚе…ій—ӯPythonдјҡиҜқгҖӮ

жіЁпјҡеҝ…йЎ»е®үиЈ…Python 2.7жҲ–3.6зүҲжң¬д»ҘдҪҝз”ЁLabVIEW PythonеҮҪж•°гҖӮе°Ҫз®ЎдёҚж”ҜжҢҒзҡ„ PythonзүҲжң¬еҸҜд»ҘдёҺLabVIEW PythonеҮҪж•°дёҖиө·дҪҝз”ЁпјҢдҪҶNIе»әи®®д»…дҪҝз”Ёж”ҜжҢҒзҡ„Python зүҲжң¬гҖӮи®ҝй—®ni.com/infoе№¶иҫ“е…ҘдҝЎжҒҜд»Јз ҒpythonпјҢдәҶи§ЈжӣҙеӨҡе…ідәҺе®үиЈ…Pythonзҡ„ дҝЎжҒҜгҖӮ

NXGйЈҺж јжҺ§д»¶ жҺ§д»¶йҖүжқҝеҢ…еҗ«ж–°зҡ„NXGйЈҺж јеүҚйқўжқҝжҺ§д»¶гҖӮдҪҝз”ЁNXGйЈҺж јзҡ„жҺ§д»¶пјҢеҲӣе»әLabVIEWNXGйЈҺж јзҡ„еүҚйқў жқҝгҖӮжҺ§д»¶зҡ„еӨ–и§ӮйҡҸз»Ҳз«Ҝз”ЁжҲ·иҝҗиЎҢVIзҡ„е№іеҸ°ж”№еҸҳгҖӮеҰӮжһңе°ҶVIиҝҒз§»еҲ°LabVIEW NXGпјҢдҪҝз”ЁиҝҷдәӣжҺ§д»¶еҸҜд»ҘжңҖеӨ§йҷҗеәҰең°еҮҸе°‘еүҚйқўжқҝзҡ„еӨұзңҹгҖӮ

LabVIEW 2018ж–°еўһдәҶй”ҷиҜҜеҜ„еӯҳеҷЁд»Ҙз®ҖеҢ–еҗҜз”ЁдәҶе№¶иЎҢеҫӘзҺҜзҡ„ForеҫӘзҺҜзҡ„й”ҷиҜҜеӨ„зҗҶгҖӮй”ҷиҜҜеҜ„еӯҳеҷЁеҸ–д»ЈдәҶе№¶иЎҢForеҫӘзҺҜдёҠй”ҷиҜҜз°Үзҡ„з§»дҪҚеҜ„еӯҳеҷЁпјҢеҰӮд»ҘдёӢзЁӢеәҸжЎҶеӣҫжүҖзӨәгҖӮ еӣҫ 3 й”ҷиҜҜеҜ„еӯҳеҷЁеҸҜиҮӘеҠЁеҗҲе№¶е№¶иЎҢеҫӘзҺҜзҡ„й”ҷиҜҜгҖӮеңЁForеҫӘзҺҜдёҠй…ҚзҪ®е№¶иЎҢеҫӘзҺҜж—¶пјҢLabVIEWе°ҶиҮӘеҠЁжҠҠз§»дҪҚеҜ„еӯҳеҷЁиҪ¬жҚўдёәй”ҷиҜҜеҜ„еӯҳеҷЁпјҢд»ҺиҖҢйҒөеҫӘйҖҡиҝҮз§»дҪҚеҜ„еӯҳеҷЁдј иҫ“й”ҷиҜҜзҡ„жңҖдҪіе®һи·өгҖӮ

й’ҲеҜ№дёҚеҗҢж•°жҚ®зұ»еһӢиҮӘе®ҡд№үиҮӘйҖӮеә”VI жҜ”иҫғйҖүжқҝж–°еўһжЈҖжҹҘзұ»еһӢеӯҗйҖүжқҝгҖӮдҪҝз”ЁвҖңжЈҖжҹҘзұ»еһӢвҖқVIе’ҢеҮҪж•°еҸҜејәеҲ¶и®©иҮӘйҖӮеә”VI(.vim)еҸӘжҺҘеҸ—ж»Ў и¶ізү№е®ҡиҰҒжұӮзҡ„ж•°жҚ®зұ»еһӢгҖӮдҪҝз”Ёзұ»еһӢдё“з”Ёз»“жһ„еҸҜдёәжҢҮе®ҡж•°жҚ®зұ»еһӢиҮӘе®ҡд№үиҮӘйҖӮеә”VI (.vim)дёӯзҡ„д»Јз Ғж®өгҖӮ
